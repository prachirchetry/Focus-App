# ⏱ Focus App

A modern Pomodoro + productivity web app with multiple focus modes, lofi music, and task tracking.

---

## 🚀 Features
- Pomodoro timer
- Break / Deep Work / Sprint modes
- Lofi + break music
- XP system
- Task manager
- Aesthetic glass UI

---

## 🖥 How to run
Just open `index.html` in a browser  
OR host it using Netlify / GitHub Pages


## 🛠 Tech Stack
- HTML
- CSS (Glass UI)
- JavaScript
